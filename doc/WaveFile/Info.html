    class WaveFile::Info

    Contains metadata about an existing wave file. Returned by WaveFile::Reader.info.

    This class is immutable - once a new Info is constructed, it canâ€™t be modified.

    Attributes

    audio_format[R]

    Returns a Fixnum indicating the audio format, such as 1 for PCM or 3 for IEEE float.

    bits_per_sample[R]

    Returns the number of bits per sample, such as 8, 16, 32, or 64.

    block_align[R]

    Returns the number of bytes in each sample frame. For example, in a 16-bit stereo file, this will be 4 (2 bytes for each 16-bit sample, times 2 channels).

    byte_rate[R]

    Returns the number of bytes contained in 1 second of sample data. Is equivalent to #block_align * sample_rate.

    channels[R]

    Returns the number of channels, such as 1 or 2.

    duration[R]

    Returns a Duration instance for the total number of sample frames in the file

    file_name[R]

    Returns the name of file this Info contains metadata about.

    sample_frame_count[R]

    Returns the total number of sample frames in the file. A sample frame contains a single sample for each channel. So if there are 1,000 sample frames in a stereo file, this means there are 1,000 left-channel samples and 1,000 right-channel samples.

    sample_rate[R]

    Returns the number of samples per second, such as 44100.
